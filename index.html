ROADSTAR CINEMATIC MOVIE

機材・ソフトウェア
Camera: Nikon Z6ii / Lens: z35mm f1.8 / Gimbal: DJI RS5

作品概要

夜の街並みとROADSTARの美しいシルエットを活かし、ブランドPVを想定した映像作品として自主制作を行いました。

作品の狙い

車そのものではなく、所有するライフスタイルの魅力を伝える。高級感と疾走感を両立した広告映像を目指しました。

制作体制

企画・撮影・編集：川上隆平 / 人数：制作2人 出演2人

工夫した点

  • 撮影前にロケハンを実施し、車両が映えるスポットを選定
  • 広告映像を意識した構図設計
  • グレーディングによる高級感の演出

制作期間

ロケハン：1日 / 撮影：1日 / 編集：2日

株式会社DYM インターン制作

実績
  • 最高3万回/1本のインプレッション
  • 3ヶ月間で約150本以上の動画制作
  • 撮影・構成・編集を一貫して担当
機材・ソフトウェア
Camera: Nikon Z6ii / Lens: z35mm f1.8

作品概要

株式会社DYM様にて長期インターンを行っている際に制作した動画のひとつです。会社が運営する飲食店のモデル動画の「撮影」「台本作成」「動画編集」を一人で行いました。また、毎日投稿を実現するために、3ヶ月間で約150本以上の動画を制作しました。

作品の狙い

8000円のビストロコースの訴求動画です。上司の提案する訴求方法に対して、最適な台本を考え、制作を行い、店舗の持つ高級感や料理の美しさを演出する工夫をしました。

工夫した点

  • 計画を立て、ライティングやカメラワークでの演出
  • 店舗の動画として、適切かつ情報分析に沿った動画の制作
  • 安っぽい印象を与えない動画のテンポや、テロップ

制作期間

撮影：約3時間 / 制作：2〜3時間/1本

制作体制

工夫した点

  • 商品の「手軽さ」を意識した、テンポの良いカット割
  • 広告映像を意識した、動きのあるカメラワーク

制作期間
